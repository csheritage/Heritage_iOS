# Heritage

> 코드스쿼드 해커톤 (2019-08-15)

***Heritage*** 는 코드스쿼드의 각 코스(모바일, 프론트, 백엔드) 의 현재 수강생, 졸업생 모두의 집단 지성을 모으기 위한 플랫폼입니다. 각 분야별 면접 질문을 공유하고, 모인 질문을 기반으로 모의 면접을 볼 수도 있습니다.
